# Dashboard de Obras

Publicacao do dashboard estatico usado no GitHub Pages.

## Como publicar

1. Configure o GitHub Pages para publicar a partir da pasta `docs`.
2. Suba a pasta `docs` completa para o repositorio.
3. Depois da publicacao, confira:
   `https://integra6a-star.github.io/dashboard-obras/dados.json`

O arquivo publicado precisa mostrar `atualizado_em` com a data mais recente gerada localmente.

## Como atualizar os dados

1. Atualize as planilhas dentro de `docs`.
2. Rode `atualizar.bat` na raiz do projeto.
3. Confira os JSONs gerados em `docs`.
4. Publique a pasta `docs` novamente.
